## Como usar

Abre el archivo `index.html` directamente en cualquier navegador. No requiere instalacion ni servidor.

## Ver en linea

Puedes publicar esta presentacion gratis con **GitHub Pages**:

1. Sube esta carpeta a un repositorio en GitHub
2. Ve a `Settings > Pages`
3. En "Source" selecciona `main` y carpeta `/root`
4. GitHub te dara un enlace publico tipo `https://tu-usuario.github.io/circular-familias/`
